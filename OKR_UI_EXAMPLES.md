## Animation Details

### Over-Achievement Animations

1. **Progress Bar Pulse**
   - Duration: 2s
   - Timing: ease-in-out
   - Effect: Gentle brightness pulse

2. **Shimmer Effect**
   - Duration: 2s infinite
   - Effect: Light streak from left to right
   - Color: White with 20% opacity

3. **Percentage Color Transition**
   - From: Pillar color
   - To: Gold (#FFD700)
   - Timing: Instant on reaching >100%

## Accessibility

- Progress bars have proper `aria-valuemin`, `aria-valuemax`, `aria-valuenow`
- All inputs have `aria-label`
- Keyboard navigation supported (Tab, Enter, Escape)
- Screen readers announce percentage and over-achievement status
- Color is not the only indicator (emoji + animation)
